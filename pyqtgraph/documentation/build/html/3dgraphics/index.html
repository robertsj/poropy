Pyqtgraph’s 3D Graphics System¶

The 3D graphics system in pyqtgraph is composed of a view widget and several graphics items (all subclasses of GLGraphicsItem) which can be added to a view widget.

Note: use of this system requires python-opengl bindings. Linux users should install the python-opengl packages from their distribution. Windows/OSX users can download from http://pyopengl.sourceforge.net.
